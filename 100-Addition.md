# Addition Checkpoint
Create a function that takes two numbers.
function numberSum() {
    
}
Your function should return the sum of the two numbers passed in.

## Examples:

### Example 1

```
console.log(yourFunction(2, 3));
```

should log 

```
5
```

### Example 2

```
console.log(yourFunction(2, 7));
```

should log


```
9
```
